# Nigeria-digital-skills-ecosystem
Systems-level analysis of digital skills in Nigeria 


## Project Overview
This project presents a systems-level analysis of digital skills 
development in Nigeria, examining how three major training programs 
— 3MTT, HNG Internship, and ALX Africa — interact within a shared 
market and how pricing, structure, and funding model affect 
long-term sustainability.

## Dataset
- Records: 1,000 rows
- Programs: 3MTT · HNG · ALX
- Period: 2022 – 2025
- Variables: 19 columns
- Cost Models: Free · Subsidized · Paid

## Tasks Completed
- Task 1: Data Aggregation & Cleaning
- Task 2: Ecosystem Causal Loop Diagram
- Task 3: Talent Supply–Demand Model
- Task 4: Pricing Distortion & Behavioral Economics
- Task 5: Economic Value Flow Simulation
- Task 6: Ecosystem Shock Simulation
- Task 7: Strategic Insights & Recommendations

## Key Findings
- Oversupply Index = 3.3x across all cost models
- Price effect on job placement = 0.000000
- Free programs capture zero revenue per enrollee
- 38,000+ graduates lost under 80% grant withdrawal shock

## Master Insight
Nigeria's digital skills ecosystem is supply-saturated 
but demand-deficient. The problem is not too few people 
being trained — it is that trained talent is not being 
absorbed into value-generating work.
